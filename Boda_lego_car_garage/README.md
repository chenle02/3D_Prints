In this folder, we will build a Lego car garage.

Model: Lego 77244 formula 1 car set.
Dimension: 8 inches (20 cm) long, 3 inches (8 cm) wide, and over 1.5 inches (4 cm) high.

Design: 
1. Front view: garage door opens upwards, with a few small windows on the top
   half part of the door.
2. No bottom, flat roof. 
3. Wall/roof width is 5mm.
4. Generate a script to generate stl file and latter we will print it on
   BambuLab printer. 5.
